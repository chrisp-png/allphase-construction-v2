## Card States

### Default State
```
┌─────────────────────────────────────┐
│  Title of Blog Post                 │
│                                     │
│  Brief excerpt describing the       │
│  content of this blog post...       │
│                                     │
│  Read Guide →                       │
└─────────────────────────────────────┘
```
- Background: Dark gray (`bg-zinc-900`)
- Border: Gray (`border-zinc-800`)
- Title: White
- Excerpt: Gray
- Link: Red

### Hover State
```
┌─────────────────────────────────────┐ ✨
│  🔴 Title of Blog Post              │
│                                     │
│  Brief excerpt describing the       │
│  content of this blog post...       │
│                                     │
│  Read Guide ➡️                       │
└─────────────────────────────────────┘
```
- Background: Dark gray (unchanged)
- Border: **Red** (`border-red-600`)
- Title: **Red** (color change)
- Excerpt: Gray (unchanged)
- Link: Red with animated arrow
- Shadow: Enhanced (`shadow-xl`)

## Responsive Breakpoints

```
Mobile (< 768px):
├─ Grid:          1 column (block)
├─ Card Width:    100%
└─ Stack:         Vertical

Tablet (≥ 768px):
├─ Grid:          2 columns (md:grid-cols-2)
├─ Card Width:    ~50% minus gap
└─ Layout:        2×2 grid

Desktop (≥ 1024px):
├─ Grid:          2 columns (maintained)
├─ Container:     Wider padding
└─ Layout:        2×2 grid with more space
```

## Animation Details

### Arrow Animation (Hover)
```
Default:  →
Hover:    ➡️  (translates 4px right)

CSS:
group-hover:translate-x-1
transition-transform
```

### Card Border Transition
```
Default:  border-zinc-800
Hover:    border-red-600

CSS:
transition-all
```

### Title Color Transition
```
Default:  text-white
Hover:    text-red-600

CSS:
transition-colors
group-hover:text-red-600
```
