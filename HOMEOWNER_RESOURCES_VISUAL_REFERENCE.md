## Animations

### Card Hover
```
Trigger:  Mouse enters card
Duration: 200ms
Easing:   ease-in-out

Changes:
├─ Border color:  zinc-800 → red-600
├─ Shadow:        none → lg with red glow
├─ Title color:   white → red-500
└─ Arrow:         translate-x-0 → translate-x-1
```

### Link Hover
```
"Read Guide" Link:
├─ Text color:    red-500 → red-400
└─ Arrow slide:   0 → 4px right

"Explore All" Link:
└─ Text color:    red-600 → red-500
```

## Card Content Structure

```
Each card contains:

1. Category Badge (top)
   ├─ Visual label
   ├─ Topic indicator
   └─ Scannable at a glance

2. Title (main)
   ├─ Full article title
   ├─ Descriptive and SEO-friendly
   └─ Clickable (entire card is link)

3. Excerpt (body)
   ├─ 1-2 line summary
   ├─ Key value proposition
   └─ Entices user to click

4. CTA (bottom)
   ├─ Action-oriented text
   ├─ Arrow for visual direction
   └─ Distinct from excerpt
```

## Hover Interaction Flow

```
1. User hovers over card
   ↓
2. Border animates to red
   ↓
3. Red glow shadow appears
   ↓
4. Title changes to red
   ↓
5. Arrow slides 4px right
   ↓
6. Visual feedback complete
   ↓
7. User clicks anywhere on card
   ↓
8. Navigate to blog post
```
